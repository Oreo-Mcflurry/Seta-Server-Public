<aside>

### 🥕 기능

- 가수의 정보를 API와 크롤링으로 가져와 MongoDB에 저장
- GET요청이 오면 가수의 정보를 Json형태로 Response


### 🔨 기술 스택 및 사용한 라이브러리

- Node.js
- MongoDB
